# Mashiv Author Coupon Price

WooCommerce plugin that allows specific coupons to set eligible products to a fixed price of 30 NIS.

## Features

- Adds a custom checkbox to WooCommerce coupons:
  - "קופון סופר - מחיר 30 ₪"
- Allows site administrators to create author-specific coupons.
- Uses WooCommerce built-in restrictions for:
  - Allowed email addresses
  - Product restrictions
- Changes the price of eligible products to 30 NIS when a marked coupon is applied.

## Usage

1. Create a WooCommerce coupon.
2. Restrict the coupon to the relevant author email address.
3. Restrict the coupon to the relevant product.
4. Enable:
   - "קופון סופר - מחיר 30 ₪"
5. Save the coupon.

When the coupon is applied, the eligible product will be priced at 30 NIS.
